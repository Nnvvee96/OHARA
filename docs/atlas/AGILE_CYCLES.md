# AGILE_CYCLES.md - Adaptive Orchestration Cycles
# PROJECT: OHARA

## 1. Cycle Structure
Cycle: Monatlicher Governance-Loop.
Phases: Intent → Orchestrate → Execute → Retro → Compound.
Gate: Phase Exit Criteria aus IMPLEMENTATION_PLAN.md.

## 2. Intent Phase
Define: Monatliche Ziele in natürlicher Sprache.
  Beispiel: "50 akzeptierte Atome, 5 Patterns in EMERGING+, Aria und Nova aktiv"
Decompose: Supervisor (Gott) bricht in Wizard-Tasks auf.
Assign: Route zu aktiven Wizards via god redirect.
Gate: Löst das echte epistemische Lücken? Bias-Check.

## 3. Orchestrate Phase
Supervisor (Gott): Setzt Ziele, approves STRUCTURAL, redirects bei Drift.
Wizards: Laufen autonom in ihren Cycles.
Human-on-Loop: Gott eskaliert nur Anomalien und STRUCTURAL-Entscheidungen.

## 4. Execute Phase
Wizard-Cycles: Stündlich (konfigurierbar via SCOUT_INTERVAL_SECONDS).
Review-Queue: Täglich durch Gott (5-10 Min via god digest).
Pattern-Promotions: Wöchentlich (30 Min).
Adversarielle Audits: Monatlich pro Domain (2h).
Tier: Gemini Flash für Extraktion; Gemini Pro für Skeptic/Reasoning.
Diagnostics: cycle_vitality Tabelle, SESSION_LOGS.md.

## 5. Retro Phase
Metriken:
  - Acceptance Rate (Ziel: 30-60%)
  - Counter-Evidence Coverage (>1 pro VALIDATED Pattern)
  - Pattern Deprecation Rate (10-20% jährlich = gesunde Zirkulation)
  - Book Staleness (<10% stale Dependencies)
  - Kosten pro akzeptiertem Atom
Shred: Fehlgeschlagene Patterns red-teamen.
Evidence: cycle_vitality Logs — keine Screenshots.

## 6. Compound Phase
Lektionen → SKILLS_COMPOUNDING.md
Neue Prinzipien → PATTERNS.md (additive only)
Technische Anti-Patterns → TECHSTACK.md
Philosophie-Updates → SOUL.md Ideen-Log
Wizard-Erkenntnisse → wizard_memory Tabelle (Phase 6+)

## 7. Verification Gates
Intent Alignment: Passt zu SOUL.md Philosophie?
Epistemic Quality: Acceptance Rate + Skeptic-Coverage.
Wizard Health: Vitality Signals, keine CRITICAL Anomalien.
God Governance: STRUCTURAL Queue reviewed, keine offenen Anomalien >48h.

Update Protocol: Neue Cycle-Strukturen via PATTERNS.md superseden.
